# Contributions

BlueTeam.News is a maintainer-led project. Unsolicited pull requests are not
accepted. Forks are welcome under the [MIT License](LICENSE).

You may open a focused bug report when the released software is demonstrably
broken, but there is no support commitment, response-time guarantee, roadmap,
or promise that a report will result in a change. Feature requests are not
accepted.

Do not report security vulnerabilities in a public issue. Use GitHub's private
vulnerability reporting flow described in [SECURITY.md](SECURITY.md).

If an exceptional external change is ever considered, the maintainer will
initiate that conversation and define the review and licensing terms for that
specific change.
